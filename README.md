# kv-elixir
Learning Elixir by implementing a key-value database.

Based on the [Introduction to Mix and OTP](https://elixir-lang.org/getting-started/mix-otp/introduction-to-mix.html) from Elixir website.

## Test
```bash
mix test
```
## Run on REPL
```bash
iex -S mix
```
## Compile
```bash
mix compile
```
